列表组

列表组是灵活又强大的插件，不仅能用于显示一组简单的元素，还能用于复杂的定制的内容。

基本实例

最简单的列表组仅仅是一个列表条目的无序列表，另外还需要设置适当的类。我们提供了一些预定义样式，你可以根据自身的需求通过 CSS 自己定制。

  • Cras justo odio
  • Dapibus ac facilisis in
  • Morbi leo risus
  • Porta ac consectetur ac
  • Vestibulum at eros

徽章

给列表组加入徽章组件，它会自动被放在右边。

  • 14 Cras justo odio

链接

用 <a> 代替标签 <li> 标签可以组成一个全是链接的列表组（还要注意的是，我们需要将 <ul> 标签替换为 <div> 标签）。没有必要给列表组中的每个元素都加一个父元素。

Cras justo odio Dapibus ac factilisis in Morbi leo risus Porta ac consectetur ac Vestibulum at eros

按钮

列表组中的元素也可以直接就是按钮（也同时意味着父元素必须是 <div> 而不能用 <ul> 了），并且无需为每个按钮单独包裹一个父元素。注意不要使用标准的 .btn 类！
