恭喜！我们的加密已经为您的域名颁发了证书！以下是您可以用于您网站的签名证书。
Nginx安装说明：
  1. 将以下证书（文本包含您的域证书和中间证书）复制并粘贴到名为“chained.pem”的文本文件中。
  2. 如果尚未完成，则生成非默认dhparams。
    openssl dhparam -out dhparam.pem 4096
  3. 将“chained.pem”和“dhparam.pem”复制到/ etc / ssl / certs /。
    scp chained.pem root@foo.com:/etc/ssl/certs/chained.pem 
scp dhparam.pem root@foo.com:/etc/ssl/certs/dhparam.pem
  4. 复制“domain.key”/ etc / ssl / private /。
    scp domain.key root@foo.com:/etc/ssl/private/domain.key
  5. 更新您的网络服务器配置以使用https（以下示例）。
    服务器{ 
    listen 443; 
    server_name foo.com; 
    ssl on; 
    ssl_certificate /etc/ssl/certs/chained.pem; 
    ssl_certificate_key /etc/ssl/private/domain.key; 
    ssl_session_timeout 5m; 
    ssl_protocols TLSv1 TLSv1.1 TLSv1.2; 
    ssl_ciphers ECDHE-RSA-AES256-GCM-SHA384：ECDHE-RSA-AES128-GCM-SHA256：DHE-RSA-AES256-GCM-SHA384：ECDHE-RSA-AES256-SHA384：ECDHE-RSA-AES128-SHA256：ECDHE-RSA -AES256-SHA：ECDHE-RSA-AES128-SHA：DHE-RSA-AES256-SHA：DHE-RSA-AES128-SHA; 
    ssl_session_cache共享：SSL：50米; 
    ssl_dhparam /etc/ssl/certs/dhparam.pem; 
    ssl_prefer_server_ciphers on; 

    location / { 
        return 200'Hello world！';
        add_header Content-Type text / plain; 
    } 
}
Apache安装说明：
  1. 将第一个证书部分（例如第一个“----- BEGIN CERTIFICATE -----”部分）复制并粘贴到名为“domain.crt”的文本文件中。
  2. 将第二个证书部分（例如第二个“----- BEGIN CERTIFICATE -----”部分）复制并粘贴到名为“intermediate.pem”的文本文件中。
  3. 将“domain.crt”和“intermediate.pem”复制到/ etc / ssl / certs /。
    scp domain.crt root@foo.com:/etc/ssl/certs/domain.crt 
scp intermediate.pem root@foo.com:/etc/ssl/certs/intermediate.pem
  4. 复制“domain.key”/ etc / ssl / private /。
    scp domain.key root@foo.com:/etc/ssl/private/domain.key
  5. 更新您的网络服务器配置以使用https（以下示例）。
    <VirtualHost _default_：443> 
        ServerName foo.com:443
